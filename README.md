# Power-of-Thor

Python code to solve the power of thor game in CodeInGame.com

https://www.codingame.com/ide/puzzle/power-of-thor-episode-1

### Input : 
[End X position] [End Y position] [Start X position] [Start Y position]
[Width of field]
[Hight of field]

#### Example : 
0 0 10 10
40
18
          
### Output :
[Direction to move]

#### Example :
N (North)
NE (North-East)
E (East)
SE (South-East)
S (South)
SW (South-West)
W (West)
NW (North-West)

### Dependencies:
- Numpy
